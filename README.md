# Petr Vitner CV



## Installation

Install XeLaTeX via TeX Live or MiKTeX:

**Windows:**
```bash
# Download and install MiKTeX from https://miktex.org/download
# or TeX Live from https://tug.org/texlive/
```

**macOS:**
```bash
brew install --cask mactex
```

**Linux:**
```bash
sudo apt-get install texlive-xetex  # Debian/Ubuntu
sudo dnf install texlive-xetex      # Fedora
```

## Compilation

Compile with **XeLaTeX** (required for custom fonts):

```bash
xelatex main.tex
```

## Modifications
